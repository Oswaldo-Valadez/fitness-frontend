# <Feature> AI prompts

## Implementation prompt requirements
- Source-of-truth files to read first
- Exact target scope
- Files that must not be touched
- Tests to add first
- Contract and generated-code boundaries
- Targeted and full verification commands
- Required completion report format

## Review prompt requirements
- Severity model: Critical/High/Medium/Low/Informational
- Explicit assumptions and missing evidence
- Verdict: APPROVED/CHANGES_REQUESTED/BLOCKED
